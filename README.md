# Introduction
This is an open source project that uses C#/Unity to reimplement Paladin III and Paladin III Gaiden (note: it is not a transplant, but a rewrite). The implementation method is to rewrite the game logic and load and read the re-implementation by using the original game resources and according to the operation mode and effect of the original game. Please read the source code for the specific implementation method.

Note: Paladin III and Paladin III Gaiden belong to the original works of Shanghai Softstar, and the copyright belongs to Softstar Technology and Daewoo Corporation. This project does not include any games in Paladin III and Paladin III Gaiden ontology data. This project follows the GPL-3.0 agreement, but it is limited to the code of this project. Any pictures, audio and video, and game data related to Legend of Sword and Fairy III or Gaiden are not in this scope.
